    <h1>EduConnect SL</h1>
    <p>A responsive educational landing page designed to improve access to digital learning in Sierra Leone.</p>

    <h2>Project Overview</h2>
    <p>
        EduConnect SL is a Digital Public Good (DPG) solution that connects students and teachers
        through an online platform. It provides access to courses, learning materials, and academic support.
    </p>

    <h2>Objectives</h2>
    <ul>
        <li>Improve access to quality education</li>
        <li>Support digital learning</li>
        <li>Create a responsive website</li>
    </ul>

    <h2>Target Users</h2>
    <ul>
        <li>Students</li>
        <li>Teachers</li>
        <li>Schools</li>
        <li>Parents</li>
    </ul>

    <h2>SDG Alignment</h2>
    <ul>
        <li>SDG 4: Quality Education</li>
        <li>SDG 10: Reduced Inequalities</li>
    </ul>

    <h2>Technologies Used</h2>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>Visual Studio Code</li>
    </ul>

    <h2>Features</h2>
    <ul>
        <li>Responsive design (Desktop, Tablet, Mobile)</li>
        <li>Hero section</li>
        <li>Feature section</li>
        <li>Call-to-action section</li>
        <li>Footer with contact details</li>
    </ul>

    <h2>Design</h2>
    <ul>
        <li>Teal background with white sections</li>
        <li>Modern buttons</li>
        <li>Clean layout based on wireframe</li>
    </ul>
